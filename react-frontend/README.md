# Smart Automated Drying System - React Frontend

A modern React-based dashboard for monitoring and controlling the Smart Automated Drying System. Built with Vite, TypeScript, Tailwind CSS, and Framer Motion for a responsive and interactive user experience.

## Features

- Real-time sensor monitoring
- Interactive dashboard with animations
- Responsive design for desktop and mobile
- Built with modern React and TypeScript

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Navigate to the react-frontend directory:
   ```bash
   cd react-frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run lint` - Run ESLint for code linting
- `npm run preview` - Preview the production build locally
